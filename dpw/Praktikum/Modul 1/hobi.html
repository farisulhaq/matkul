Assalamualaikum wr.wb

Perkenalkan nama saya ahmad farisul haq bisa di panggil faris.

Saya akan menceritakan tentang hoby saya.

Hobi saya saat masih smp adalah bermain catur dan berolahraga,

saya biasa main catur pas malam dan setelah pulang sekolah,

saya seneng bermain catur solanya terpengaruh lingkungan,

banyak yang main catur dan terbiasa untuk berkontribusi saat waktu senggang.

namun pas sma hobi saya bermain game online,

menurut saya hobi tergantung kebiasaan,

jika terbiasa maka akan dianggap hobi oleh saya.

Hobi awal tetap saya senangi namun secara virtual bermainnya.

Untuk hari ini hobi saya masih ngilang ngilang soalnya sibuk di perkuliahan.

Terima kasih telah membaca hobi saya, sekian terima kasih.
